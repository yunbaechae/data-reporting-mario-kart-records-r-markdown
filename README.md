# R Markdown Reporting and Data Visualization for Mario Kart World Records

STAT1378 Coding and Communication in Statistics Assignment 2 (2021)

This assignment focused on reproducible data reporting and visualization of Mario Kart 64 world records. Created R Markdown report and custom R package for trend exploration and documentation.

## Key Tasks
- Analyzed records by track, player, time, duration, and shortcuts  
- Visualized performance trends and record history  
- Developed R package for data processing and reporting
